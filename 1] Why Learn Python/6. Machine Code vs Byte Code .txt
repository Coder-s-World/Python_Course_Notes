
----Machine Code----
Machine code is directly understood by the computer’s CPU.	
It runs directly on hardware.	
It is very fast.	
It is machine-dependent.	
Generated after final compilation.	
Example: C language compiled output.	

----Byte Code----
Byte code is an intermediate code understood by a virtual machine.
It needs a virtual machine like JVM to run.
It is slower than machine code.
It is platform-independent.
Generated before machine code.
Example: Java bytecode (.class files).
